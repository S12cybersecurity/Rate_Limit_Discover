# Rate_Limit_Discover
HTTP/HTTPS Discover of Rate Limit Discover

**Installation**

git clone https://github.com/S12cybersecurity/Rate_Limit_Discover/

pip3 install requests

pip3 install argparse

**Usage**

python3 rate_limit.py --url url
